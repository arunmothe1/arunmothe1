## 📂 Featured Project

### 🚀 Highly Available Web Application Deployment on AWS Using Terraform

Designed and deployed a highly available and scalable 3-tier web application infrastructure on Amazon Web Services (AWS) using Terraform as the Infrastructure as Code (IaC) tool.

#### Key Responsibilities
- Designed a secure Virtual Private Cloud (VPC) with public and private subnets across multiple Availability Zones.
- Provisioned EC2 instances using Launch Templates for application deployment.
- Configured an Application Load Balancer (ALB) to distribute incoming traffic across multiple instances.
- Implemented Auto Scaling Groups (ASG) to ensure high availability and automatic scaling based on workload.
- Created and managed Security Groups following the principle of least privilege.
- Configured IAM roles and policies to provide secure access to AWS resources.
- Deployed and managed an Amazon RDS database for reliable and managed data storage.
- Monitored infrastructure using Amazon CloudWatch for performance and health monitoring.
- Automated the complete infrastructure deployment using reusable Terraform modules and configuration files.
- Used Git and GitHub for version control and collaborative project management.

#### AWS Services & Tools Used
- **AWS:** EC2, VPC, IAM, S3, RDS, Auto Scaling, Application Load Balancer (ALB), CloudWatch, Security Groups
- **Infrastructure as Code:** Terraform
- **DevOps Tools:** Git, GitHub, Docker
- **Operating System:** Linux
